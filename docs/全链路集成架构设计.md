## 二、批量模拟引擎设计

### 2.1 引擎架构

```python
class BatchSimulationEngine:
    """
    批量模拟引擎
    
    职责：
    1. 加载画像批次
    2. 为每个画像绑定问卷和情境
    3. 并发执行模拟
    4. 收集和存储回答
    5. 监控进度和成本
    """
    
    def __init__(self, config: SimulationConfig):
        self.config = config
        self.llm_client = LLMClient()
        self.cost_tracker = CostTracker()
        self.progress_tracker = ProgressTracker()
    
    async def run_batch(
        self,
        personas: list[Persona],
        questionnaire: Questionnaire,
        scenario_contexts: dict[str, ScenarioContext]
    ) -> BatchSimulationResult:
        """
        执行批量模拟
        
        参数：
            personas: 消费者画像列表
            questionnaire: CBC问卷（选择集）
            scenario_contexts: 各群体的情境上下文
        
        返回：
            BatchSimulationResult: 包含所有回答和元数据
        """
        
        # 创建任务队列
        tasks = []
        for persona in personas:
            scenario = scenario_contexts[persona.segment]
            task = self._simulate_persona(persona, questionnaire, scenario)
            tasks.append(task)
        
        # 并发执行（控制并发度）
        semaphore = asyncio.Semaphore(self.config.max_concurrency)
        
        async def bounded_simulate(task):
            async with semaphore:
                return await task
        
        # 执行并收集结果
        results = await asyncio.gather(*[
            bounded_simulate(t) for t in tasks
        ], return_exceptions=True)
        
        # 处理结果
        successful = [r for r in results if not isinstance(r, Exception)]
        failed = [r for r in results if isinstance(r, Exception)]
        
        return BatchSimulationResult(
            total=len(personas),
            successful=len(successful),
            failed=len(failed),
            responses=successful,
            cost_summary=self.cost_tracker.get_summary(),
            duration_seconds=self.progress_tracker.get_duration()
        )
    
    async def _simulate_persona(
        self,
        persona: Persona,
        questionnaire: Questionnaire,
        scenario: ScenarioContext
    ) -> PersonaResponse:
        """
        模拟单个消费者的完整问卷作答
        """
        responses = []
        
        # 构建模拟Agent的上下文
        sim_context = self._build_sim_context(persona, scenario)
        
        for choice_set in questionnaire.choice_sets:
            # 构建当前题目的Prompt
            prompt = self._build_choice_prompt(
                persona=persona,
                scenario=scenario,
                choice_set=choice_set,
                history=responses[-3:]  # 最近3题的历史
            )
            
            # 调用LLM
            with self.cost_tracker.track(
                agent="SIM",
                model=self.config.sim_model,
                persona_id=persona.id
            ):
                response = await self.llm_client.call(prompt)
            
            # 解析响应
            parsed = self._parse_choice_response(response)
            responses.append(parsed)
            
            # 疲劳度调整
            if len(responses) > 8:
                scenario.fatigue_level = "high"
        
        return PersonaResponse(
            persona_id=persona.id,
            segment=persona.segment,
            responses=responses,
            completion_time=datetime.utcnow()
        )
```

### 3.2 数据流状态机

```
[INIT] → 用户提交研究需求
  │
  ▼
[PERSONA_GENERATING] → GEN Agent生成画像
  │  └─ 审校不通过 → [PERSONA_FIXING] → 重新生成
  │
  ▼
[QUESTIONNAIRE_BUILDING] → 构建CBC问卷
  │  └─ 设计效率不足 → [DESIGN_OPTIMIZING] → 重新设计
  │
  ▼
[SIMULATING] → 批量模拟作答
  │  ├─ 进度更新（每10%）
  │  └─ 成本告警 → [DEGRADED] → 切换轻量模型
  │
  ▼
[ANALYZING] → 自动触发分析
  │  └─ 模型不收敛 → [MODEL_RETRYING] → 增加采样
  │
  ▼
[REPORTING] → 生成洞察报告
  │
  ▼
[COMPLETED] → 通知用户
  │
  ▼
[ARCHIVED] → 数据归档（90天后）
```
