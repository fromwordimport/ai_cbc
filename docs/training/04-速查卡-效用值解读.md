# 速查卡：效用值解读

> **版本**：v1.0
> **定位**：单页速查，快速查阅效用值解读要点
> **编写者**：小培

---

## 一句话定义

**效用值 = 某个属性水平对消费者的"吸引力分数"**

---

## 符号含义

| 符号 | 含义 |
|------|------|
| **+** | 正向吸引力（消费者喜欢） |
| **-** | 负向吸引力（消费者不喜欢） |
| **绝对值大** | 影响强烈 |
| **绝对值小** | 影响微弱 |

---

## 三种比较方式

### 1. 水平间比较（同一属性）

```
价格：¥2999(+1.2) > ¥3999(+0.3) > ¥4999(-0.8)

→ 消费者明显偏好低价
→ 从¥3999涨到¥4999，痛苦感更强（降幅更大）
```

### 2. 属性间比较（不同属性）

```
"选西门子(+0.6)而非美的(-0.3)" = 效用提升 +0.9
"价格从¥4999降到¥3999" = 效用提升 +1.1

→ 对该消费者，降价¥1000的吸引力 > 品牌升级
```

### 3. 个体间比较（不同消费者）

```
消费者A价格系数：-0.005（极度敏感）
消费者B价格系数：-0.001（相对不敏感）

→ A是"价格驱动型"，B是"价值驱动型"
```

---

## 关键注意事项

| 注意点 | 说明 |
|--------|------|
| 效用值是相对的 | 只有比较才有意义，+2不比+1"好两倍" |
| 关注符号和排序 | 比具体数值更重要 |
| 价格系数应为负 | 为正则模型/数据有问题 |
| 编码方式影响数值 | Effects vs Dummy，数值不同但排序一致 |

---

## 快速判断表

| 观察 | 结论 |
|------|------|
| 某水平效用最高 | 消费者最喜欢 |
| 某水平效用最低 | 消费者最不喜欢 |
| 水平间差异大 | 偏好强烈 |
| 水平间差异小 | 不太敏感 |
| 价格系数为负 | ✅ 正常 |
| 价格系数为正 | ❌ 异常，检查模型 |
