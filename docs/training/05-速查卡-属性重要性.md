# 速查卡：属性重要性

> **版本**：v1.0
> **定位**：单页速查，快速查阅属性重要性解读要点
> **编写者**：小培

---

## 一句话定义

**属性重要性 = 该属性各水平效用的极差 / 所有属性极差之和**

它回答：**消费者做选择时，最看重什么？**

---

## 计算公式

```
属性重要性 = 该属性极差 / 所有属性极差之和

其中：极差 = 最大效用 - 最小效用
```

---

## 计算示例

```
价格：   +1.5 ~ -1.5    极差 = 3.0
容量：   +1.0 ~ -0.8    极差 = 1.8
品牌：   +0.9 ~ -0.3    极差 = 1.2
功能：   +0.7 ~ -0.5    极差 = 1.2

总极差 = 3.0 + 1.8 + 1.2 + 1.2 = 7.2

价格重要性 = 3.0 / 7.2 = 41.7%
容量重要性 = 1.8 / 7.2 = 25.0%
品牌重要性 = 1.2 / 7.2 = 16.7%
功能重要性 = 1.2 / 7.2 = 16.7%
```

---

## 重要性等级解读

| 重要性 | 含义 | 业务行动 |
|--------|------|---------|
| > 40% | **决定性因素** | 优先优化，资源倾斜 |
| 20%-40% | **重要因素** | 不可忽视，持续投入 |
| 10%-20% | **次要因素** | 差异化空间 |
| < 10% | **边缘因素** | 投入需谨慎 |

---

## 群体差异示例

| 群体 | 价格 | 容量 | 品牌 | 功能 | 洞察 |
|------|------|------|------|------|------|
| 精致白领 | 35% | 20% | **25%** | 20% | 品牌敏感，愿付溢价 |
| 新手宝妈 | 30% | **35%** | 15% | 20% | 容量刚需 |
| Z世代租客 | **55%** | 15% | 10% | 20% | 极度价格敏感 |

---

## 关键注意事项

| 注意点 | 说明 |
|--------|------|
| 重要性之和 = 100% | 但单个属性不能单独解读 |
| 重要性是相对的 | 取决于其他属性的极差 |
| 群体差异大 = 细分机会 | 不要"一刀切"策略 |
| 与预期不符时 | 检查价格区间、水平设定 |

---

## 快速判断表

| 观察 | 结论 |
|------|------|
| 某属性 > 40% | 决定性因素 |
| 某属性 < 10% | 边缘因素，谨慎投入 |
| 群体间差异 > 15% | 强烈细分信号 |
| 与预期排序不符 | 检查设计或样本 |
