## 一、数据收集合规检查

## 三、审计日志合规检查（BE-7 接口重点）

### 3.1 日志内容要求

| 字段 | 要求 | 检查方法 | 状态 |

| 操作时间戳 | ISO 8601 格式，UTC | 检查日志 schema | 待检查 |
| 操作类型 | CREATE / READ / UPDATE / DELETE / GENERATE | 枚举值校验 | 待检查 |
| 操作者 | 用户ID / Agent ID / 系统 | 身份标识 | 待检查 |
| 数据对象 | 批次ID / 画像ID / 实验ID | 关联标识 | 待检查 |
| 操作结果 | SUCCESS / FAILURE / PARTIAL | 结果记录 | 待检查 |

| 成本记录 | 对应成本跟踪ID | 检查与小控成本系统集成 | 待检查 |

| 检查项 | 要求 | 检查方法 | 状态 |
