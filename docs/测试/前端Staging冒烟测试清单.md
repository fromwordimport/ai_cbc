## 七、已知 Staging 限制（非缺陷）

1. **数据隔离**：Staging 与生产数据独立，可能只有 demo 数据
2. **分析任务**：Staging 可能使用 mock 分析，收敛诊断为固定值
3. **画像数量**：Staging 预置画像数量有限
4. **LLM 调用**：Staging 可能关闭真实 LLM 调用以控制成本
