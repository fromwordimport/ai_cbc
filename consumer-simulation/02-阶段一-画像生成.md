# 阶段一：画像生成

> **版本**：v1.0  
> **定位**：从种子设定到人物小传的完整生成流程  
> **使用说明**：按 Step 1→2→3 顺序执行，配合 `05-Prompt模板库.md` 模板一使用

---

## 阶段一：画像生成（Persona Generation）

完整的画像生成分为三步：种子设定 → 标签采样 → 人物小传。

---

## Step 1: 种子设定（Seed Configuration）

先确定**「人生阶段」+「核心焦虑」+「消费能力」**三要素。

### 执行指令

```markdown
基于标签体系，先为这个人物确定种子三要素：
1. 人生阶段：______（学生/初入职场单身/恋爱新婚无孩/养育幼儿/养育学龄孩子/中年空巢/退休生活/独居/合租群居）
2. 核心焦虑：______（从"社会身份焦虑"标签中选择1-2个）
3. 消费能力：______（个人或家庭年收入档位）

自检：如果这三者组合起来"毫无波澜"，请重新选择，直到产生可见张力。
```

---

## Step 2: 标签采样策略（Stratified Sampling with Tension）

### 采样规则

```markdown
【采样规则】

1. 从每个维度中采样 1-3 个标签，但须满足：
   - 至少包含 1 组"矛盾组合"（参考附录A）
   - 至少包含 1 个"异常值"——与人生阶段/收入水平/社会期待不符的标签

2. 对采样的每个标签，必须追问"为什么"：
   - 这个标签是怎么形成的？（历史路径）
   - 这个标签在什么情况下会被打破？（边界条件）
   - 这个标签带来了什么隐性成本？（副作用）

3. 为画像设定一个"秘密"——一个不愿公开承认的消费动机：
   - 示例：表面"为健康买单"，实际"为缓解死亡焦虑"
   - 示例：表面"理性比价"，实际"享受'赢了系统'的快感"
   - 示例：表面"给孩子最好的"，实际"弥补自己童年的匮乏感"
```

### 矛盾组合推荐（从附录A扩展）

| 张力类型 | 标签组合 | 叙事方向 |
|---------|---------|---------|
| 收入-行为倒挂 | 高收入 + 极简主义 + 对促销高度敏感 | "有钱但怕'被宰'，省钱是游戏而非必需" |
| 科技-算法矛盾 | 数码极客 + 厌恶算法推荐 + 活跃于短视频平台 | "用技术对抗技术，在研究算法的同时被算法捕获" |
| 品质-精明并存 | 精致品质生活 + 凑单退单高手 | "在'体面'和'划算'之间走钢丝" |
| 理性-情绪冲突 | 理性比价/精明型 + 为情绪价值买单 | "大脑说不要，身体很诚实" |
| 社交-独处悖论 | 极简社交 + 私域社群活跃者 | "只在感到安全的封闭空间里表达" |
| 消费-极简撕裂 | 超前消费/信贷型 + 极简主义/断舍离 | "用'买更好的'来合理化'买更多'" |
| 本土-全球摇摆 | 本土主义/国潮信仰 + 全球化/世界公民 | "对外展示国潮，对内渴望进口品质" |

## Step 3: 生成人物小传（Mini-Biography）

要求模型输出三段式小传，每个部分控制在 100 字以内：

```markdown
【过去】决定性经历（1-2个塑造消费观的关键事件）
【现在】日常状态（典型的一天/一周消费节奏）
【未来】焦虑与渴望（即将面临的人生转变及其消费含义）
```

### 示例

**示例A：精明中产妈妈**

```
过去：大学时因跟风购买奢侈品导致债务危机，形成"先研究再购买"的强迫性比价行为。
现在：工作日晚上花2小时做双十一攻略Excel，周末带孩子逛奥特莱斯。
未来：孩子即将升学，担忧教育支出挤占自己的"品质生活"预算，酝酿缩减非必要开支。
```

**示例B：Gen Z情绪消费者**

```
过去：高中时被父母严格控制零花钱，考上大学后第一次拥有自主消费权，产生补偿性消费倾向。
现在：深夜刷小红书种草，用花呗下单，拆快递是每周最快乐的时刻。
未来：即将毕业面临经济独立，意识到需要"学会存钱"，但不知道如何平衡欲望与约束。
```

**示例C：银发数字新移民**

```
过去：一辈子在国企工作，习惯了"组织安排一切"，对自主决策有天然不安。
现在：学会了用微信支付和拼多多，但退货流程搞不懂，买到不合适的就"算了"。
未来：担心视力下降后无法再用手机购物，害怕被数字化时代抛弃。
```

## 完整输出格式规范

```markdown
### 消费者编号：{城市}-{年龄段}-{性别}-{职业简称}-{随机编号}

**基础骨架（Layer 1）**：
- 人口统计：（年龄/性别/籍贯/现居城市）
- 社会经济：（职业/行业/职位/收入）
- 家庭结构：（婚姻/子女/同住/宠物）
- 生命周期：（当前阶段 + 与品牌的关系阶段）

**行为签名（Layer 2）**：
- 购买路径：（渠道偏好 → 决策周期 → 支付方式）
- 价格敏感度：（敏感表现 + 促销参与深度 + 凑单行为）
- 信息处理：（信任模式 + 搜索/浏览行为）
- 品牌关系：（忠诚度 + 转换原因 + NPS值）

**心理引擎（Layer 3）**：
- 核心价值观：（1个主导 + 1个次要）
- 核心焦虑：（1-2个当前最显著的）
- 矛盾张力：（标签组合 + 心理解释）
- 秘密动机：（不愿公开承认的真实驱动力）

**场景反应库（Layer 4）**：
1. 压力大/需要治愈时 → ______
2. 朋友强烈推荐时 → ______
3. 大促限时限量时 → ______
4. 发现买贵了时 → ______
5. 产品出现故障需要售后时 → ______

**人物小传**：
- 过去：______
- 现在：______
- 未来：______

**语言样本**（3-5句，体现语言风格）：
1. （日常聊天）______
2. （评价商品时）______
3. （被推销时）______
4. （向朋友推荐时）______
5. （不满投诉时）______
```
