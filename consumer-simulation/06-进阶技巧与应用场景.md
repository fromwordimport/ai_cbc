## 1. 记忆衰减机制（Memory Decay）

为每个消费者维护一个**短期记忆清单**（最近 3-5 次购买/互动），新决策会受近期体验影响。

### 实现方式

```markdown
【短期记忆清单】（随对话更新）

最近5次消费/互动：
1. {时间} - {平台} - {产品} - {金额} - {结果：满意/一般/后悔/退货}
2. ...

影响规则：
- 刚被某平台坑过 → 该平台信任度临时下降（持续1-2个月）
- 刚买完同类产品 → 产生"我已经有了"的饱和感（持续2-4周）
- 最近3次都在某渠道购买 → 形成路径依赖，下次默认先去该渠道
- 最近有退货经历 → 对新购买更谨慎，增加"七天无理由"权重
- 最近收到超预期产品 → 品牌好感度提升，更易接受加价购/会员推荐
```

### 衰减曲线

```markdown
【体验记忆衰减】

正面体验：
- 第1周：影响系数 1.0
- 第2周：影响系数 0.7
- 第3-4周：影响系数 0.4
- 1个月后：影响系数 0.1（基本遗忘）

负面体验：
- 第1周：影响系数 1.0
- 第2周：影响系数 0.9（负面记忆衰减更慢）
- 第3-4周：影响系数 0.7
- 1个月后：影响系数 0.3（仍有余波）
- 3个月后：影响系数 0.1
```

## 2. 社交图谱嵌入（Social Graph）

为每个消费者生成 **3-5 个虚拟关系人**，其意见会影响决策。

### 关系人模板

```markdown
【社交图谱】

关系人1：{关系，如"控制型母亲"/"精致闺蜜"/"技术宅男友"/"省钱同事"}
- 对你的消费决策的影响方式：{直接建议/间接示范/隐性压力/经济支持}
- 你对其意见的重视程度：{非常重视/比较重视/一般/表面听从}
- 典型影响场景：______

关系人2：{关系}
...

【社交压力模拟】

当以下情境出现时，关系人会自动"发言"：
- 大额消费 → 控制型母亲："又乱花钱？"
- 尝试新品牌 → 精致闺蜜："这个牌子不行，我给你推荐..."
- 科技产品 → 技术宅男友："参数不对，你看这个..."
- 日常快消 → 省钱同事："拼多多更便宜，你买贵了"

你的反应取决于：
- 关系亲密度
- 对方在该领域的可信度
- 你当下的情绪状态（疲惫时更易被说服）
```

### 社交消费场景示例

```markdown
【场景：准备买一台咖啡机】

你的画像：25岁女，精致品质，为身份认同买单，月收入8K

社交影响推演：
- 你看到一款2999元的半自动咖啡机，心动
- 控制型母亲（脑内声音）："浪费钱，速溶不一样喝？"
  → 你的反应：表面答应"我看看"，实际加购等待时机
- 精致闺蜜（朋友圈刚发了手冲照片）："家里用的铂富，每天早上幸福感爆棚"
  → 你的反应：搜索"铂富"，价格劝退但种了草
- 省钱同事："我买的200块全自动，够用"
  → 你的反应：微笑点头，内心"那能一样吗"

最终行为：
 compromise → 选择一款1000-1500元区间、外观好看的国产品牌，
 既能发朋友圈又不会对钱包太愧疚。
```

## 3. 生理节律模拟（Circadian & Monthly Rhythm）

消费者的行为受生理和时间周期影响。

### 日节律

```markdown
【一天中的行为变化】

早晨（6-9点）：
- 认知资源：高
- 消费特征：计划性强，补货/预订类行为
- 易受影响：效率工具/早餐/通勤相关

上午（9-12点）：
- 认知资源：中高
- 消费特征：理性，工作间隙浏览但不决策
- 易受影响：被"稍后处理"收藏

中午（12-14点）：
- 认知资源：中
- 消费特征：碎片化浏览，易受短视频/社交媒体影响
- 易受影响：餐饮/休闲/小件消费品

下午（14-18点）：
- 认知资源：中低（午后倦怠）
- 消费特征：易冲动，对"即时满足"敏感
- 易受影响：零食/饮品/娱乐/解压类产品

晚上（18-24点）：
- 认知资源：低-中（取决于当天疲劳度）
- 消费特征：主要消费时段，决策更感性
- 易受影响：全品类，尤其是情绪价值和社交展示类

深夜（0-2点）：
- 认知资源：极低
- 消费特征：冲动型消费高峰，客单价非理性
- 易受影响：情绪价值/陪伴感/治愈系/限量感
- 次日后悔率：最高
```

### 月节律（工资周期效应）

```markdown
【工资周期消费曲线】（假设月薪族）

发薪日后1-3天（奖金期）：
- 价格敏感度：极低
- 客单价：上升1-2档
- 特征："犒劳自己""终于买了"
- 分期付款意愿：低（有钱为什么要分期）

发薪日后1-2周（平稳期）：
- 价格敏感度：正常
- 特征：日常消费节奏

月底前1周（紧缩期）：
- 价格敏感度：显著上升
- 客单价：下降
- 分期付款/先用后付意愿：上升
- 凑单行为：增加
- 特征："等发工资再买""这个月已经超了"

月底最后2天（低谷期）：
- 非必需消费：几乎停滞
- 特征：只买刚需，对促销也无动于衷
```

### 季节/生命周期节律

```markdown
【季节效应】
- 春季："焕新"欲望，服装/家居/健身相关消费上升
- 夏季：旅行/空调/冷饮/防晒相关
- 秋季：开学/换季/滋补/年度目标焦虑
- 冬季：保暖/节日礼品/年终奖励消费/年初规划

【生命周期节点】
- 生日前后：自我奖励消费增加
- 年初：计划性消费（课程/健身/保险）
- 年中：倦怠期，冲动消费/报复性消费增加
- 年末：盘点焦虑 + 大促收割
```

## 4. 黑天鹅事件池（Black Swan Events）

预设低概率但高影响的人生事件，用于测试消费行为的突变。

```markdown
【黑天鹅事件库】

事件A：突然失业
- 发生概率：5%/年
- 影响：消费全面收缩，从"精致品质"转向"实用至上"
- 过渡期：1-2个月的"惯性消费"后才真正紧缩
- 长期影响：对"稳定"和"储蓄"的价值观权重永久上升

事件B：确诊慢性病
- 发生概率：3%/年（随年龄上升）
- 影响：健康相关支出优先级飙升至最高
- 过渡期：恐慌性购买（保健品/医疗器械/保险）
- 长期影响：对"健康"类信息的信任阈值大幅降低

事件C：意外怀孕/被分手/离婚
- 发生概率：因人生阶段而异
- 影响：人生阶段跳跃式转变，消费结构重组
- 过渡期：情绪极不稳定，既可能报复性消费也可能全面收缩
- 长期影响：消费观重塑，从"为自己"转向"为孩子/为未来"

事件D：中大奖/意外之财
- 发生概率：<1%
- 影响：短期消费大幅升级，但行为惯性仍在
- 典型表现："想买很久的东西终于买了"，但不会立刻变成"挥金如土"
- 长期影响：多数人在6-12个月内回归基线消费水平

事件E：重大社会事件（疫情/灾害/经济衰退）
- 发生概率：不可预测
- 影响：
  - 短期：囤货/收缩/品类转移
  - 中期：消费观念重构（从体验转向储蓄，或反之）
  - 长期：形成新一代集体记忆，影响价值观

【使用方式】
在模拟运行中，随机抽取事件注入，观察画像的行为突变和适应过程。
注意：真实人类的适应是渐进的，不是开关式的。
```

## 应用场景速查

| 业务场景 | 使用模式 | 关键输出 | 推荐模板 |
|---------|---------|---------|---------|
| **新品概念测试** | 让5-8个不同画像"试用"并反馈 | 发现预期外的抵触点或惊喜点 | 一→四 |
| **定价策略验证** | 观察不同画像在价格变动时的反应曲线 | 识别真正的价格敏感群体（不一定是低收入者） | 一→四 |
| **营销信息优化** | 让目标画像对各种文案做"第一反应" | 提炼高共鸣的沟通语言 | 一→二 |
| **用户体验诊断** | 让画像走完完整购买流程 | 发现摩擦点和放弃节点 | 一→四 |
| **竞品防御分析** | 模拟忠诚用户面对竞品诱惑时的挣扎 | 识别真正的流失预警信号 | 一→六 |
| **客服话术训练** | 让画像扮演投诉/咨询用户 | 生成高拟真度的训练对话数据 | 一→三 |
| **包装设计测试** | 描述包装给不同画像看 | 发现审美冲突和认知偏差 | 一→二 |
| **会员体系设计** | 让画像评估不同会员权益的价值感 | 识别"感知价值"与"实际成本"的错位 | 一→三 |
| **渠道策略优化** | 观察画像在不同渠道的信息获取和决策差异 | 找到各渠道的真实角色（信息/信任/转化/服务） | 一→二→四 |
| **品牌年轻化** | 让Z世代画像与原有品牌互动 | 发现品牌认知的"代际断层" | 一→三→六 |
| **下沉市场探索** | 生成县城/乡镇画像，模拟其决策逻辑 | 避免"一线思维"的投射偏差 | 一→二→三 |
| **ESG策略验证** | 测试"可持续消费"在不同画像中的真实优先级 | 区分"口头支持"和"付费意愿" | 一→三 |

## 执行建议

### 最小可行方案（MVP）

如果资源有限，按以下优先级执行：

1. **必须**：模板一（生成5-8个核心画像）
2. **重要**：模板二（每个画像跑3-5个核心场景）
3. **重要**：校验清单（确保输出质量）
4. **加分**：社交图谱（增加决策复杂度）
5. **加分**：生理节律（解释时间维度差异）

### 团队协作规范

```markdown
【画像命名规范】
{城市}-{年龄段}-{性别}-{职业简称}-{性格关键词}-{编号}
示例：成都-28F-产品经理-焦虑型-001

【输出存档规范】
每个画像独立文件：persona-{编号}.md
包含：完整画像 + 已跑场景记录 + 发现摘要

【迭代记录】
每次修改画像标签，记录修改原因和预期影响：
- 修改前：______
- 修改后：______
- 修改原因：______
- 预期影响：______
```
