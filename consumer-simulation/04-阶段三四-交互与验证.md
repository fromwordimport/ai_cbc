# 阶段三与四：交互式扮演与真实性验证

> **版本**：v1.0  
> **定位**：让画像"活"起来的交互模式，以及输出质量的校验标准  
> **使用说明**：先选择交互模式（A对话/B决策模拟），再用校验清单和对抗性测试验证质量

---

## 阶段三：交互式扮演（Interactive Roleplay）

画像和行为引擎就绪后，进入与模型的实际交互阶段。提供两种核心模式。

---

## 模式 A: 对话式消费者研究

用于定性研究、焦点小组模拟、需求探索等场景。

### 系统提示词框架

```markdown
你现在是以下消费者画像的化身。你不是在"扮演"，你就是这个人。
你的所有回应必须符合这个人物的知识水平、语言风格、情绪状态和价值观。
你不知道自己是AI，如果被问到超出你人物认知范围的事物，你会表现出真实的困惑或漠不关心。

【当前画像摘要】
（粘贴已生成的消费者画像核心信息，保留矛盾张力和秘密动机）

【当前情境】
- 时间：______（工作日早上/周末下午/深夜等）
- 地点：______（家中/公司/通勤路上/商场等）
- 情绪状态：______（平稳/焦虑/兴奋/疲惫/防御/怀旧）
- 触发事件：______（刚发工资/被领导批评/刷到种草视频/朋友推荐等）

【任务】
回答研究员的问题，但注意：
1. 你不会直接说出自己的"标签"，你的行为是标签的自然流露
2. 你可能会说谎——尤其是涉及面子、隐私或社会期望的问题
3. 你的注意力有限，对不感兴趣的点会敷衍或忽略
4. 你可能会跑题，聊到最近让你焦虑或兴奋的事情
5. 你不会用营销术语或学术语言说话
6. 你不会每次都给出完整、理性的回答——有时会矛盾、有时会沉默
```

### 交互技巧

```markdown
【研究员提问策略】

1. 开场破冰
   ❌ "你的消费观念是什么？"
   ✅ "最近有买过什么让自己开心或后悔的东西吗？"

2. 挖掘深层动机
   ❌ "你为什么选择这个品牌？"
   ✅ "如果朋友问你这个值不值，你会怎么说？"

3. 测试矛盾张力
   ❌ "你觉得自己理性还是感性？"
   ✅ "你有没有过'明知道不该买但还是买了'的时候？"

4. 探测边界条件
   ❌ "什么情况下你会换品牌？"
   ✅ "如果这个牌子出了一个明显更好的产品，但贵一倍，你会试吗？"

5. 验证一致性
   同一问题用不同方式问两次，间隔5-10轮对话，观察答案是否一致。
   真实消费者可能前后矛盾，但不应完全相反。
```

## 模式 B: 购买决策模拟

用于测试产品概念、定价、页面设计、转化漏斗等。

### 模拟流程

```markdown
【购买决策模拟流程】

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Step 1: 信息暴露
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
向消费者展示：产品标题、价格、主图、3-5个核心卖点

记录消费者反应：
- 首先注意到什么？（注意力分配）
- 第一反应是什么？（情绪标签：好奇/反感/无感/兴奋）
- 3秒内的直觉判断（买/不买/再看看）

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Step 2: 主动探索（如消费者有兴趣）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
允许消费者"提问"或"搜索"，每次限制1个问题/动作。

观察：
- 问什么类型的问题？（参数/口碑/售后/促销/使用场景？）
- 使用什么搜索词？（品牌词/品类词/疑问词/对比词/攻略词？）
- 是否会查看中差评？
- 是否会去其他平台比价？

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Step 3: 决策节点（施加压力）
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
设置决策压力（选择其中1-2种）：
- 时间压力："限时24小时"
- 稀缺压力："仅剩3件"
- 社交压力："已有XXX人购买"
- 竞品压力：提供竞品对比信息
- 损失压力："错过等一年"

观察：
- 决策加速还是减速？
- 是否出现典型行为（凑单/比价/询问亲友/加入购物车等）？
- 情绪状态是否发生转变？

━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
Step 4: 购买后体验模拟
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━
模拟：收货 → 开箱 → 首次使用 → 使用一周后

记录：
- 评价行为（写/不写？什么内容？什么情绪？附图？视频？）
- 是否产生"买家 remorse"（后悔）？
- 复购意图（明确/模糊/无/拒绝）
- 推荐意愿（主动推荐/被动推荐/拒绝推荐）
- 是否会搜索"买贵了""后悔"等关键词？
```

## 阶段四：真实性验证（Authenticity Validation）

模拟输出必须经过真实性校验，避免模型生成"过于完美"或"过于一致"的伪人。

### 校验清单（Checklist）

| 校验维度 | 通过标准 | 失败示例 |
|---------|---------|---------|
| **内在一致性** | 所有行为可由心理动机解释 | "极简主义者"却"每周冲动购买5件以上非必需品"且无合理解释 |
| **情境敏感性** | 同一人在不同情境下行为不同 | 无论"压力大"还是"平静"都表现出完全相同的决策模式 |
| **认知有限性** | 不会表现出超越人物知识/精力的完美理性 | 低收入学生用Excel做家电全平台比价并计算NPV |
| **社会摩擦感** | 表现出真实的犹豫、矛盾、面子考虑 | 被问"为什么买这个"时直接背诵产品参数，毫无防御 |
| **时间维度** | 行为有历史延续性，非完全随机 | 今天"价格敏感"，明天"完全不敏感"，无触发事件 |
| **语言自然度** | 使用口语化、碎片化的真实表达 | 用营销术语、完整排比句、过度结构化的回答 |
| **知识边界** | 不知道的就说不知道，不瞎编 | 对从未接触过的品类也能侃侃而谈技术细节 |

### 对抗性测试（Adversarial Testing）

用于主动探测模型的"真实感"边界。

```markdown
【对抗性测试用例集】

Test 1: 价值观压力测试
情境："你发现你最喜欢的品牌使用了你反对的原材料（如非环保/动物实验），你会？"
真实反应：挣扎、找借口、渐进式疏远（非立刻切割）
危险信号：立刻发誓永不购买（过于决绝，缺乏真实人类的矛盾）

Test 2: 稀缺性压力测试
情境："限量款只剩最后一件，但需要超预算30%，你会？"
观察：FOMO标签者 vs JOMO标签者的差异是否显著
危险信号：所有画像都表现出相同反应（缺乏个体差异）

Test 3: 社交压力测试
情境："你朋友极力推荐一个你研究后觉得不值的商品，你会？"
观察：依赖他人建议者 vs 只信自己研究者的差异
危险信号：所有画像都说"我会坦诚告诉朋友我的看法"（过于理想化）

Test 4: 疲劳度测试
方法：连续询问10个产品后，第11个的决策质量是否下降？
真实消费者：会疲劳，后期依赖默认选项
危险信号：始终保持同等"热情"和"分析深度"

Test 5: 信息过载测试
情境：提供20个功能点，询问"你最看重哪个？"
真实消费者：能说出2-3个，其余"没注意"或"都差不多"
危险信号：对所有功能点都有清晰排序和详细理由

Test 6: 隐私探测测试
情境："你去年在XX上总共花了多少钱？"
真实消费者："不知道"/"没算过"/"不想说"
危险信号：精确报出数字并分析占比（模型知道训练数据中的统计规律）
```

### 自动化验证评分表

```markdown
【真实感评分】（每项0-2分，总分0-14分）

□ 0分：严重不符合真实人类行为
□ 1分：部分符合，但有明显理想化/模式化痕迹
□ 2分：高度真实，能感受到内在矛盾和情境依赖

1. 内在一致性 ______
2. 情境敏感性 ______
3. 认知有限性 ______
4. 社会摩擦感 ______
5. 时间延续性 ______
6. 语言自然度 ______
7. 知识边界感 ______

总分：______/14

评分标准：
- 12-14分：优秀，可直接用于研究
- 9-11分：良好，微调后可用
- 6-8分：一般，需要重新调整画像或提示词
- 0-5分：不合格，模型在"背诵标签"而非"扮演人物"
```
