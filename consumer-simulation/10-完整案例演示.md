# 完整案例演示：从业务问题到消费者洞察

> **版本**：v1.0  
> **定位**：端到端展示"定义问题 → 生成画像 → 模拟情境 → 验证质量 → 提取洞察"的完整流程  
> **使用说明**：跟随本案例一步步执行，可复现完整的研究过程

---

## 案例背景

**业务方**：某国产护肤品牌"植颜"  
**产品**：新品精华油，主打"以油养肤""熬夜急救"  
**定价**：299元/30ml（中端价位）  
**目标**：上市前验证产品概念、定价接受度、核心卖点有效性  
**研究方法**：用 AI 模拟真实消费者，观察其面对产品信息时的真实反应

---

## Step 0：定义研究问题

在生成画像之前，先明确要回答什么问题：

```markdown
【核心问题】
1. 谁会买？—— 目标人群画像
2. 为什么买？—— 核心购买动机
3. 什么阻止购买？—— 关键阻力点
4. 定价是否敏感？—— 299元在不同人群中的接受度
5. 哪个卖点最打动人？—— 注意力分配和决策转折点
```

**研究设计**：生成 3 个差异化画像，分别模拟"看到产品信息 → 产生兴趣 → 决策过程"的完整路径。

## Step 1：生成目标画像（使用 02 + 05 模板一）

### 1.1 确定种子策略

根据研究问题，需要覆盖三类核心人群：

| 画像 | 种子组合 | 研究价值 |
|------|---------|---------|
| A | 初入职场单身 + 同辈压力 + 年收入5-8万 | 价格敏感型，验证299元是否门槛过高 |
| B | 养育幼儿 + 育儿焦虑 + 家庭年收入25-50万 | 品质导向型，验证"成分安全"卖点 |

（粘贴 consumer-profile.md 全文）

1. 每个画像必须包含1组矛盾组合
2. 为每个标签写出"形成原因"和"被打破的边界条件"
3. 设定一个"秘密动机"
4. 写出"过去-现在-未来"三段式小传
5. 提供3-5句语言样本

【输出格式】

## Step 5：提取业务洞察

### 5.1 单画像洞察

**画像A（小林）—— "情绪补偿型"消费者**
- **核心动机**："今天被批评了，需要对自己好一点"
- **关键触点**：博主信任 + 限量紧迫感
- **阻力点**：价格（月入5000 vs 299）+ 已有库存过多
- **转化策略**：
  - 限时折扣/分期付款降低当下痛感
  - "买正装送小样+试用装"降低试错成本
  - 强调"每天只需10元"的日摊成本

**画像B（陈姐）—— "安全求证型"消费者**
- **核心动机**："成分必须安全，我不能冒险"
- **关键触点**：成分透明度 + 第三方检测报告
- **阻力点**：需要足够的"社会证明"（大量真实用户评价）
- **转化策略**：
  - 详情页突出成分表 + 美丽修行安全评分
  - 提供"敏感肌实测""孕妈可用"等背书
  - 大促期间凑满减（她喜欢"算到最优"的感觉）

**画像C（王总）—— "效率极简型"消费者**
- **核心动机**："少即是多，一瓶解决所有问题"
- **关键触点**："精简护肤"概念 + 权威推荐（教练/医生）
- **阻力点**：对新品牌缺乏信任，需要长期观察
- **转化策略**：
  - 强调"一瓶替代精华+面霜"的效率价值
  - 提供试用装或小规格版本降低首次尝试门槛
  - 男性护肤内容营销（他可能在男性护肤信息茧房外）

### 5.2 跨画像共性洞察

```markdown
【共性发现】

1. "限量"是双刃剑
   - 画像A：限量直接触发冲动购买（FOMO）
   - 画像B：限量让她警惕"是不是营销套路"
   - 画像C：限量完全无感（"我不赶时间"）
   → 建议：限量策略只针对冲动型人群，对理性人群强调"长期供应"

2. "以油养肤"概念需要翻译
   - 画像A：理解为"用完会发光"（效果导向）
   - 画像B：理解为"天然成分"（安全导向）
   - 画像C：理解为"简化步骤"（效率导向）
   → 建议：同一概念在不同人群中的认知不同，需要分层沟通

3. 299元定价的接受度分层
   - 画像A：偏贵，但如果能降低单次使用成本（"每天10元"）可接受
   - 画像B：合理，但期待赠品/满减（"要让我觉得占了便宜"）
   - 画像C：不贵，但要求"值得"（成分、功效、品牌背书）
   → 建议：统一标价299，但针对不同人群设计不同的价值传递方式

4. 购买路径完全不同
   - 画像A：内容电商（小红书/抖音）→ 冲动下单
   - 画像B：货架电商（天猫/京东）→ 研究后等大促
   - 画像C：品牌官网/线下 → 长期观察后决策
   → 建议：全渠道布局，但各渠道的沟通重点不同
```

## 关键成功因素

1. **画像差异化足够**：如果三个画像反应趋同，说明画像设计失败，需要重新生成
2. **情境具体**："周三晚上11点刚被批评"比"晚上"有效100倍
3. **产品信息统一**：三个画像看到的产品信息必须一致，否则对比无效
4. **验证不走过场**：13/14分的输出和8/14分的输出，洞察质量天差地别
5. **洞察可行动**：每个洞察必须对应一个具体的业务行动建议
