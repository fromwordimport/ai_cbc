## 二、MNL 模型（基线）

### 2.1 模型公式

条件Logit模型的选择概率：

```
P(i|C) = exp(V_i) / Σ_j exp(V_j)

其中：
V_i = β₁X₁ᵢ + β₂X₂ᵢ + ... + βₖXₖᵢ

X_ki = 选项 i 在属性 k 上的取值
β_k  = 属性 k 的效用系数（待估计）
```

### 2.2 实现代码

```python
import pandas as pd
import statsmodels.api as sm
from statsmodels.discrete.discrete_choice import ConditionalLogit

class MNLModel:
    """
    多项Logit基线模型
    
    使用 statsmodels 的 ConditionalLogit 实现
    适用于快速获得聚合层面的偏好估计
    """
    
    def __init__(
        self,
        data: pd.DataFrame,
        feature_cols: list[str],
        resp_id_col: str = "resp_id",
        task_id_col: str = "task_id",
        choice_col: str = "choice"
    ):
        self.data = data
        self.feature_cols = feature_cols
        self.resp_id_col = resp_id_col
        self.task_id_col = task_id_col
        self.choice_col = choice_col
        
        self.model = None
        self.results = None
    
    def fit(self, method: str = "bfgs", maxiter: int = 100) -> "MNLModel":
        """
        拟合模型
        
        Parameters
        ----------
        method : str
            优化方法："newton", "nm", "bfgs", "lbfgs", "powell", "cg", "ncg"
        maxiter : int
            最大迭代次数
        """
        # 构建设计矩阵
        X = self.data[self.feature_cols]
        y = self.data[self.choice_col]
        
        # 添加常数项（如果需要）
        # X = sm.add_constant(X)
        
        # 构建分组索引（每个选择集为一组）
        groups = self.data.groupby([self.resp_id_col, self.task_id_col]).ngroup()
        
        # 使用 ConditionalLogit
        self.model = ConditionalLogit(endog=y, exog=X, groups=groups)
        
        self.results = self.model.fit(
            method=method,
            maxiter=maxiter,
            disp=False
        )
        
        return self
    
    def get_coefficients(self) -> pd.DataFrame:
        """获取系数估计表"""
        if self.results is None:
            raise RuntimeError("请先调用 fit()")
        
        params = self.results.params
        conf_int = self.results.conf_int()
        
        return pd.DataFrame({
            "parameter": params.index,
            "coef": params.values,
            "std_err": self.results.bse.values,
            "z": self.results.tvalues.values,
            "p_value": self.results.pvalues.values,
            "ci_lower": conf_int.iloc[:, 0].values,
            "ci_upper": conf_int.iloc[:, 1].values,
            "significant": self.results.pvalues.values < 0.05
        })
    
    def get_model_fit(self) -> dict:
        """获取模型拟合指标"""
        return {
            "log_likelihood": self.results.llf,
            "null_log_likelihood": self.results.llnull,
            "mc_fadden_r2": self.results.prsquared,
            "cox_snell_r2": 1 - (self.results.llf / self.results.llnull),
            "aic": self.results.aic,
            "bic": self.results.bic,
            "n_obs": self.results.nobs,
            "n_params": len(self.results.params),
            "converged": self.results.mle_retvals.get("converged", False)
        }
    
    def predict_probabilities(self, new_data: pd.DataFrame = None) -> pd.Series:
        """预测选择概率"""
        if new_data is None:
            new_data = self.data
        
        X = new_data[self.feature_cols]
        return self.results.predict(X)
```

### 2.3 参数调优建议

| 参数 | 默认值 | 调整建议 |
|------|--------|---------|
| `method` | "bfgs" | 大数据量用 "lbfgs"，收敛困难用 "powell" |
| `maxiter` | 100 | 不收敛时增至 500 |
| `cov_type` | "nonrobust" | 异方差时用 "HC1" |

### 2.4 常见问题

| 问题 | 原因 | 解决方案 |
|------|------|---------|
| 完全分离 | 某属性完美预测选择 | 合并水平或使用 Firth 惩罚 |
| 不收敛 | 初始值远离最优 | 尝试不同初始值或优化方法 |
| 标准误过大 | 多重共线性 | 检查属性相关性，删除冗余属性 |

## 三、Hierarchical Bayes（HB）—— 核心模型

### 3.1 模型公式

混合Logit的层级贝叶斯实现：

```
个体层面：
  U_ij = βᵢ' X_ij + ε_ij
  βᵢ ~ N(μ, Σ)          # 个体系数服从多元正态

总体层面（超参数）：
  μ ~ N(μ₀, A)          # 超先验
  Σ ~ IW(ν, V)          # 逆Wishart先验（或 LKJ + 半正态）

选择概率：
  P(i|C, βᵢ) = exp(βᵢ'Xᵢ) / Σⱼ exp(βᵢ'Xⱼ)
```

### 3.2 PyMC 实现

```python
import pymc as pm
import pytensor.tensor as pt
import arviz as az

class HBModel:
    """
    分层贝叶斯 Mixed Logit 模型
    
    使用 PyMC 5.x 实现，NUTS 采样器
    """
    
    def __init__(
        self,
        data: pd.DataFrame,
        feature_cols: list[str],
        resp_id_col: str = "resp_id",
        task_id_col: str = "task_id",
        choice_col: str = "choice"
    ):
        self.data = data
        self.feature_cols = feature_cols
        self.resp_id_col = resp_id_col
        self.task_id_col = task_id_col
        self.choice_col = choice_col
        
        self.n_features = len(feature_cols)
        self.model = None
        self.trace = None
        
        self._preprocess()
    
    def _preprocess(self):
        """预处理数据，构建任务索引"""
        # 受访者映射
        self.resp_ids = self.data[self.resp_id_col].unique()
        self.n_resp = len(self.resp_ids)
        self.resp_map = {rid: i for i, rid in enumerate(self.resp_ids)}
        
        # 构建任务列表
        self.tasks = []
        for (resp, task), group in self.data.groupby([self.resp_id_col, self.task_id_col]):
            X_task = group[self.feature_cols].values
            y_task = group[group[self.choice_col] == 1].index[0] - group.index[0]
            resp_idx = self.resp_map[resp]
            
            self.tasks.append({
                "X": X_task,
                "y": y_task,
                "resp_idx": resp_idx
            })
        
        self.n_tasks = len(self.tasks)
    
    def build_model(
        self,
        mu_prior: tuple = (0, 10),      # (mean, sd)
        sigma_prior: str = "half_normal", # "half_normal" or "exponential"
        lkj_eta: float = 2.0            # LKJ 先验的 eta 参数
    ):
        """
        构建 PyMC 模型
        
        Parameters
        ----------
        mu_prior : tuple
            总体均值先验 (mean, sd)
        sigma_prior : str
            总体标准差先验类型
        lkj_eta : float
            LKJ 先验参数，越大越偏向单位矩阵
        """
        self.model = pm.Model()
        
        with self.model:
            # ── 总体超参数 ──
            mu = pm.Normal(
                "mu",
                mu=mu_prior[0],
                sigma=mu_prior[1],
                shape=self.n_features,
                initval=np.zeros(self.n_features)
            )
            
            # 使用非中心化参数化提升采样效率
            # 标准差
            if sigma_prior == "half_normal":
                sigma = pm.HalfNormal("sigma", sigma=2, shape=self.n_features)
            else:
                sigma = pm.Exponential("sigma", lam=1, shape=self.n_features)
            
            # 相关矩阵（LKJ 先验）
            chol, corr, stds = pm.LKJCholeskyCov(
                "chol",
                n=self.n_features,
                eta=lkj_eta,
                sd_dist=pm.Exponential.dist(1.0, shape=self.n_features),
                compute_corr=True
            )
            
            # 协方差矩阵
            cov = pm.Deterministic("cov", chol @ chol.T)
            
            # ── 个体系数（非中心化参数化）──
            z = pm.Normal("z", mu=0, sigma=1, shape=(self.n_resp, self.n_features))
            beta = pm.Deterministic("beta", mu + (chol @ z.T).T)
            
            # ── 似然函数 ──
            log_probs = []
            
            for task in self.tasks:
                X = task["X"]                    # (n_alts, n_features)
                resp = task["resp_idx"]
                chosen = task["y"]
                
                # 效用计算
                utilities = pm.math.dot(X, beta[resp])  # (n_alts,)
                
                # log-softmax（数值稳定）
                log_prob = pm.math.log_softmax(utilities)[chosen]
                log_probs.append(log_prob)
            
            # 总对数似然
            pm.Potential("log_likelihood", pm.math.sum(log_probs))
    
    def fit(
        self,
        n_draws: int = 1000,
        n_tune: int = 1000,
        n_chains: int = 4,
        target_accept: float = 0.9,
        **kwargs
    ) -> az.InferenceData:
        """
        运行 MCMC 采样
        
        Parameters
        ----------
        n_draws : int
            每链采样数
        n_tune : int
            预烧期（tuning）迭代数
        n_chains : int
            并行链数
        target_accept : float
            NUTS 目标接受率（0.8-0.95）
        
        Returns
        -------
        az.InferenceData : 后验样本
        """
        with self.model:
            self.trace = pm.sample(
                draws=n_draws,
                tune=n_tune,
                chains=n_chains,
                target_accept=target_accept,
                return_inferencedata=True,
                **kwargs
            )
        
        return self.trace
    
    def convergence_diagnostics(self) -> dict:
        """
        收敛诊断
        
        Returns
        -------
        dict : 诊断指标
        """
        if self.trace is None:
            raise RuntimeError("请先调用 fit()")
        
        # R-hat
        rhat = az.rhat(self.trace, var_names=["mu", "sigma", "beta"])
        rhat_max = float(rhat.max().values)
        
        # ESS
        ess_bulk = az.ess(self.trace, var_names=["mu", "sigma", "beta"], method="bulk")
        ess_tail = az.ess(self.trace, var_names=["mu", "sigma", "beta"], method="tail")
        
        return {
            "rhat_max": rhat_max,
            "rhat_by_param": {
                name: float(rhat.sel({list(rhat.coords.keys())[0]: name}).values)
                for name in rhat.coords[list(rhat.coords.keys())[0]].values
            },
            "ess_bulk_min": float(ess_bulk.min().values),
            "ess_tail_min": float(ess_tail.min().values),
            "converged": rhat_max < 1.1,
            "reliable_ess": float(ess_bulk.min().values) > 400
        }
    
    def get_population_parameters(self) -> pd.DataFrame:
        """
        获取总体参数的后验摘要
        
        Returns
        -------
        pd.DataFrame : 总体均值和标准差
        """
        summary = az.summary(self.trace, var_names=["mu", "sigma"])
        return summary
    
    def get_individual_utilities(self) -> pd.DataFrame:
        """
        获取个体层面效用估计（后验均值）
        
        Returns
        -------
        pd.DataFrame : (n_resp, n_features)
        """
        beta_posterior = self.trace.posterior["beta"]
        beta_mean = beta_posterior.mean(dim=["chain", "draw"]).values
        
        return pd.DataFrame(
            beta_mean,
            index=self.resp_ids,
            columns=self.feature_cols
        )
    
    def get_individual_distribution(self, resp_id: str = None) -> pd.DataFrame:
        """
        获取特定受访者的后验分布
        
        Parameters
        ----------
        resp_id : str
            受访者ID，None 则返回所有
        
        Returns
        -------
        pd.DataFrame : 后验样本
        """
        beta_posterior = self.trace.posterior["beta"]
        
        if resp_id is not None:
            resp_idx = self.resp_map[resp_id]
            samples = beta_posterior.sel({"beta_dim_0": resp_idx})
        else:
            samples = beta_posterior
        
        # 展平链和样本维度
        return samples.stack(sample=("chain", "draw")).to_dataframe()
```

### 3.3 先验设定建议

| 场景 | mu_prior | sigma_prior | lkj_eta | 说明 |
|------|----------|-------------|---------|------|
| 默认 | (0, 10) | half_normal | 2.0 | 弱信息先验，通用 |
| 价格敏感已知 | (-0.001, 0.01) | half_normal | 2.0 | 价格系数应为负且小 |
| 强品牌差异 | (0, 5) | exponential | 1.0 | 允许更大的异质性 |
| 保守估计 | (0, 5) | half_normal | 5.0 | 降低异质性预期 |

### 3.4 MCMC 调参指南

| 症状 | 原因 | 解决方案 |
|------|------|---------|
| R-hat > 1.1 | 未收敛 | 增加 n_tune（2000+），降低 target_accept |
| ESS 过低 | 采样效率低 | 增加 n_draws，检查参数相关性 |
| 发散（divergences） | 后验几何复杂 | 增加 target_accept 到 0.9-0.95 |
| 采样极慢 | 参数空间高维 | 使用更弱的先验，考虑变量标准化 |
| 树深度超限 | 后验有细长区域 | 增加 max_treedepth |

### 4.1 实现方式一：基于 HB 效用聚类

```python
from sklearn.mixture import BayesianGaussianMixture
from sklearn.preprocessing import StandardScaler

class LatentClassClustering:
    """
    基于 HB 个体效用的高斯混合聚类
    """
    
    def __init__(self, individual_utilities: pd.DataFrame):
        self.util = individual_utilities
        self.scaler = StandardScaler()
        self.util_scaled = self.scaler.fit_transform(self.util.values)
    
    def fit_range(self, min_classes: int = 2, max_classes: int = 6) -> pd.DataFrame:
        """拟合多个类别数，返回比较表"""
        results = []
        
        for n in range(min_classes, max_classes + 1):
            model = BayesianGaussianMixture(
                n_components=n,
                covariance_type="full",
                n_init=10,
                max_iter=500,
                random_state=42
            )
            model.fit(self.util_scaled)
            
            results.append({
                "n_classes": n,
                "log_likelihood": model.lower_bound_,
                "aic": model.aic(self.util_scaled),
                "bic": model.bic(self.util_scaled),
                "converged": model.converged_
            })
        
        return pd.DataFrame(results)
    
    def get_optimal_model(self, criterion: str = "bic") -> BayesianGaussianMixture:
        """根据准则选择最优模型"""
        results = self.fit_range()
        best_n = results.loc[results[criterion].idxmin(), "n_classes"]
        
        model = BayesianGaussianMixture(
            n_components=int(best_n),
            covariance_type="full",
            n_init=10,
            max_iter=500,
            random_state=42
        )
        model.fit(self.util_scaled)
        
        return model
```

## 五、Mixed Logit（MXL）

### 5.1 与 HB 的区别

| 特性 | HB | MXL |
|------|-----|-----|
| 估计方法 | MCMC 后验采样 | 最大模拟似然（MSL） |
| 输出 | 完整后验分布 | 点估计 + 标准误 |
| 个体效用 | 可直接导出 | 需要后验推导 |
| 速度 | 慢（分钟到小时） | 中等（秒到分钟） |
| 适用场景 | 小样本、需个体效用 | 大样本、快速探索 |

## 六、模型比较与选择

### 6.1 信息准则

```python
class ModelComparator:
    """模型比较器"""
    
    @staticmethod
    def compare(models: dict[str, Any]) -> pd.DataFrame:
        """
        比较多个模型
        
        Parameters
        ----------
        models : dict
            {name: model_instance}
        
        Returns
        -------
        pd.DataFrame : 比较表
        """
        comparisons = []
        
        for name, model in models.items():
            comp = {"model": name}
            
            if hasattr(model, "get_model_fit"):
                fit = model.get_model_fit()
                comp.update({
                    "log_likelihood": fit.get("log_likelihood"),
                    "aic": fit.get("aic"),
                    "bic": fit.get("bic"),
                    "n_params": fit.get("n_params"),
                    "n_obs": fit.get("n_obs"),
                    "mc_fadden_r2": fit.get("mc_fadden_r2")
                })
            
            if hasattr(model, "convergence_diagnostics"):
                diag = model.convergence_diagnostics()
                comp["converged"] = diag.get("converged", "N/A")
                comp["rhat_max"] = diag.get("rhat_max", "N/A")
            
            comparisons.append(comp)
        
        df = pd.DataFrame(comparisons)
        
        # 标记推荐模型（最低 BIC）
        if "bic" in df.columns:
            df["recommended"] = df["bic"] == df["bic"].min()
        
        return df
```

### 6.2 选择建议

| 场景 | 推荐模型 | 理由 |
|------|---------|------|

## 八、常见问题排查手册

### 8.1 模型不收敛

```
症状：R-hat > 1.1，trace plot 多链不重合

排查步骤：
1. 检查数据质量（缺失值、异常值）
2. 增加 n_tune（建议 1000-2000）
3. 提高 target_accept（0.9 → 0.95）
4. 简化模型（减少属性或水平）
5. 检查先验设定是否合理
6. 标准化连续变量
```

### 8.2 价格系数为正

```
症状：价格系数估计值为正（违背经济学直觉）

可能原因：
1. 属性编码错误（如价格使用了 dummy coding）
2. 存在完全分离（某些水平从不被选择）
3. 样本量过小，估计不稳定
4. 属性间存在强共线性

解决方案：
1. 检查编码方式，价格应使用连续编码
2. 合并低频水平
3. 增加样本量或使用更强先验
4. 检查属性相关性，删除冗余
```

### 8.3 WTP 计算异常

```
症状：WTP 值极大或出现无穷大

原因：价格系数接近 0（分母过小）

解决方案：
1. 检查价格系数是否显著不为 0
2. 对价格系数使用较强的负向先验
3. 计算 WTP 时过滤价格系数接近 0 的个体
4. 使用群体层面的 WTP 而非个体层面
```

### 8.4 市场份额预测不合理

```
症状：份额之和不为 100%，或某产品份额为 0/100%

排查：
1. 检查是否包含 "都不选" 选项
2. 检查效用值是否过大（溢出）
3. 使用数值稳定的 softmax 计算
4. 校准 "都不选" 选项的外部效度
```
